# 🚲 Bike Sales Interactive Dashboard (Excel)

## Project Overview
This project involves a comprehensive data cleaning and visualization process for a bike sales dataset containing 1,000 records. The goal was to identify key trends in customer behavior to help a retail business optimize its marketing strategy.

## Technical Skills Demonstrated
* **Data Cleaning**: Handled missing values, standardized categorical entries (e.g., 'M' to 'Married'), and **successfully identified and removed duplicate records** to ensure data accuracy.
* **Feature Engineering**: Utilized **Nested IF Statements** to create custom `Age Brackets` for demographic segmentation.
* **Pivot Tables & Dashboards**: Built complex pivot tables and connected them to **Interactive Slicers**, allowing for dynamic filtering by Region, Education, and Marital Status.

## Key Findings
* The **0-1 mile commute range** is the primary "Hot Zone" for sales.
* **Middle-aged professionals** represent the most consistent buyer persona.
* Higher income levels correlate with a higher probability of purchase across both genders.
